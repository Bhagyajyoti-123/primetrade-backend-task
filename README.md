
# PrimeTrade Backend Task

This project was created as part of the PrimeTrade Backend Task submission.  
It includes a backend API and a simple frontend interface to interact with the API.

## Features
- Node.js + Express backend
- MySQL database integration
- JWT authentication
- API for user login and task management
- Simple frontend for API interaction

## Technologies Used
- Node.js
- Express.js
- MySQL
- HTML
- CSS
- JavaScript
- Git & GitHub

## Installation

1. Clone the repository

git clone https://github.com/Bhagyajyoti-123/primetrade-backend-task.git

2. Navigate to project folder

cd primetrade-backend-task

3. Install dependencies

npm install

4. Start the server

node server.js

Server will run on:
http://localhost:3000
